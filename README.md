# Vega: [Discord](https://discord.com) bot maker

## What is it
**Vega** is a discord bot creator based on [Discord.js](https://discord.js.org) that allows you to create your own application simply using [Nodejs](https://nodejs.org)

## How it work
Choose a name, profile picture and get your **app token** from the [Discord developer portal](https://discord.com/developers/applications/) then configure it as you wish with the interface.

--------------------------------------------------------

A server is set up to receive requests to create a project containing the information of the applications created and to put the bot online using an interface that allows generating desktop applications from HTML files.
